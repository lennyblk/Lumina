Lumina
----------------------------------------------------------------------
Développement d’un plateformer en C,
Conception d’une architecture modulaire pour gérer les états du jeu
(menu, gameplay, sauvegardes).
Implémentation d’un système de gestion de fichiers de configuration
(lecture et écriture).
Optimisation des performances et gestion des collisions dans un
environnement 2D.
Utilisation de bibliothèque externe SDL2
